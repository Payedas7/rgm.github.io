RGM Mobile Screens

A visual tour of key flows with context for each screen.

Tap an image to zoom in. Each card highlights the intent of the screen and what the user accomplishes there.

Onboarding: Basics

Collects name, gender, DOB, and spiritual status to derive targets and personalize the journey.

Onboarding: Status

Lets users declare Initiated/Ashraya/Volunteer/Missionary to map appGroup and devotional goals.

Onboarding: DOB

Date picker ensures valid DOB, driving youth vs adult targeting and consent logic downstream.

Onboarding: Confirmation

Summarizes input before moving to contact and mentor flows, reducing friction from mis-entry.

Contact & Location

Email, phone/WhatsApp, address, and country-seeded dial code; user can override the prefix freely.

Education & University

Searchable university sheet with an “Other” path for schools not listed; supports custom entry.

Mentor Selection

Active mentors ordered by sort_order from Supabase; required to proceed with Ashraya guidance.

Guide Selection

Optional guide for Krishna Life participants; tied to appGroup and Ashraya mentoring.

Children Profiles

Add child profiles with relation, DOB, grade, and Ashraya participation; under-13 consent captured.

Child Details

Structured fields for relation/gender/DOB/grade; supports Ashraya flags and targets youth goals.

Consent & Ashraya

Under-13 consent toggle and Ashraya participation switch to set correct kid/youth targets.

Practice: Japa & Arati

Daily practice card with Japa toggle/time, Mangal Arati, and Guru Puja switches, tied to targets.

Svadhyaya

Reading/hearing steppers and book picker; kirtan toggle appears when targets include kirtan minutes.

Seva

Log seva hours with breakdown by service type loaded from Supabase; contributes to seva progress.

Overview Rings

Progress rings for rounds, association, and seva; association sums read+hear+kirtan capped to target.

Calendar

Vaishnava day feed from edge function with fallback; notifications scheduled for observances.

Profile

Displays mentor, goals, and Ashraya status for the selected profile; switch profiles via dropdown.

Settings

Notifications, legal links, support, sign out, and MFA controls in one place.

MFA TOTP Setup

Shows QR + URI + secret with copy/share actions; verify code to enable; conflict-safe enrollment.

MFA Enabled

Post-verification state with option to disable; ensures MFA is active across sessions.
